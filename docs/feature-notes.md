# 기능 노트

## 프로젝트명

카리스뷰티

## 프로젝트 목적

샐리 원장님의 SMP 시술 브랜딩과 교육 브랜딩을 정리하고,
고객 신뢰 확보와 교육 전환을 위한 브랜드/콘텐츠 구조를 만드는 프로젝트

---

## 핵심 방향

- 자연스러운 SMP 브랜딩
- 하이브리드 SMP 솔루션 차별화
- 후기/비포애프터/대화캡처 기반 신뢰 확보
- 고객용/교육용 메시지 분리
- 추후 랜딩 및 전환 구조 확장 가능하게 설계

---

## 핵심 자산

### 1. 원장 소개 자산

- SMP 전문가
- 교육자
- 하이브리드 SMP 솔루션 보유
- 4년 이상 임상
- 캐나다, 창원 등 외부 지역 고객 유입

### 2. 차별성 자산

- DOT + LINE 통합 방식
- 자연스러운 결과
- 삭발 / 무삭발 전체 커버 가능
- 외모를 넘어 자신감 회복 중심 철학

### 3. 공감 자산

- 원장님 본인도 지친 1인 기술자 경험
- 단가 경쟁, 예약 피로, 현타 경험
- 직접 극복한 성공 지도 보유

### 4. 증빙 자산

- 비포/애프터 사진
- 메신저 대화 캡처
- 후기 자료
- 실제 사례 사진

---

## 주요 페이지 방향

### 1. 원장 소개 페이지

- 샐리 원장님의 스토리
- 시술 철학
- 교육 사명
- 신뢰 형성용 메시지

### 2. 시술 소개 페이지

- 하이브리드 SMP 설명
- DOT와 LINE 차이
- 자연스러운 결과 강조
- 비포/애프터 중심

### 3. 교육 소개 페이지

- 어떤 원장님에게 필요한 교육인지
- 단가 경쟁/예약 피로 공감
- 성공 지도/교육 가치 전달

### 4. 후기 페이지

- 비포/애프터
- 대화 캡처
- 실제 고객 반응
- 지역/사례 기반 신뢰 강화

---

## 이번 단계에서 할 것

1. 프로젝트 구조 세팅
2. 브랜드 메시지 정리
3. 자료 분류 구조 정리
4. 페이지 방향 정의
5. 추후 랜딩/전환 전 개발 준비 상태 만들기

---

## 이번 단계에서 하지 않을 것

- 상세 UI 개발
- 광고 집행
- 자동화 기능 개발
- 퍼널 로직 구현

---

## 추후 개발 우선순위

1. 원장 소개 + 시술 차별성 메인 페이지
2. 비포/애프터 및 후기 중심 증빙 페이지
3. 교육 전환용 소개 페이지
4. 문의/상담 연결 구조
