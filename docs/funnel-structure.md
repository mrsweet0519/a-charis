# 퍼널 구조 메모

## 핵심 흐름

### 고객용 퍼널

브랜드 신뢰 형성
→ 시술 차별성 이해
→ 비포/애프터 및 후기 확인
→ 상담/문의 전환

### 교육용 퍼널

1인 기술자 공감
→ 점 시술 한계 인식
→ 하이브리드 SMP 차별성 이해
→ 성공 지도/교육 가치 인식
→ 교육 문의 전환

---

## 자료 활용 기준

### 신뢰 형성 자료

- 원장 소개
- 프로필 이미지
- 실제 활동 스토리

### 결과 증빙 자료

- 비포/애프터 사진
- 후기 사진
- 대화 캡처

### 차별성 설명 자료

- 하이브리드 SMP 설명
- DOT + LINE 차이
- 삭발/무삭발 전체 커버 사례

### 교육 전환 자료

- 지친 1인 기술자 공감 문구
- 성공 지도 메시지
- 교육 사명/비전

---

## 페이지 우선순위

1. 원장 소개/브랜드 메인
2. 시술 차별성 소개
3. 후기/증빙 페이지
4. 교육 전환 페이지
5. 문의 연결 페이지
