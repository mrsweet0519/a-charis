# 카리스뷰티

## 개요

카리스뷰티는 SMP 전문가이자 교육자인 샐리 원장님의
시술 브랜딩과 교육 브랜딩을 정리하는 프로젝트입니다.

## 프로젝트 목적

- 자연스러운 SMP 브랜딩
- 하이브리드 SMP 솔루션 차별성 전달
- 후기/비포애프터/대화캡처 기반 신뢰 확보
- 교육 전환 구조 마련

## 현재 단계

- 1차 세팅 단계
- 구조 정리 및 기준 문서 세팅 중심
- 상세 개발 및 랜딩 구축은 추후 진행

## 핵심 자산

- 원장 소개 스토리
- 하이브리드 SMP 메시지
- 비포/애프터 사진
- 메신저 대화 캡처
- 교육 전환 메시지
